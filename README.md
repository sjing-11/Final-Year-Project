# Final-Year-Project
**Business Process Automation for Inventory Management : Purchase Reordering, Supplier Management &amp; Notification Module**
This project automates inventory management for SMEs by streamlining key processes such as purchase reordering, supplier management, and notifications.

Contributions:<br>
**I.Purchase Reordering Module**: Developed an automated system to generate purchase orders when stock reaches predefined thresholds, ensuring timely restocking. <br>
**II.Supplier Management Module**: Designed and implemented a module to manage supplier records, automate purchase order processing, and track order histories.<br>
**III.Notification Module**: Integrated a real-time notification system to alert users about stock issues, PO approvals, and other critical events.

**Technologies Used**<br>
Backend: PHP, AWS Lambda, Amazon RDS (MySQL)<br>
Cloud Infrastructure: AWS (EC2, RDS, S3, Lambda, SNS, CloudWatch)<br><br>
